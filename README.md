# SQU Internship 

This repository contains my internship report and documentation completed during my IT internship at the Information Systems Department, Sultan Qaboos University.

## Internship Overview

- **Organization:** Sultan Qaboos University (SQU) - Information Systems Department  
- **Internship Period:** 15/06/2025 - 24/07/2025  
- **Duration:** 6 Weeks  

The internship provided practical exposure to IT infrastructure, networking, cybersecurity, system administration, and automation concepts. It helped bridge the gap between academic knowledge and real-world IT environments.

## Topics Covered

During the internship, I gained hands-on experience in:

- Network configuration and troubleshooting
- IP addressing and VLAN setup
- Switches, routers, and network cabling fundamentals
- Network traffic analysis using Wireshark
- Active Directory user and group management
- PowerShell scripting fundamentals
- Linux server management
- Virtual machines and remote server access
- Cybersecurity fundamentals
- AI automation using n8n
